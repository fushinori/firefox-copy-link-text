# Copy Link Text

An extremely simple add-on for Firefox that lets you copy the link text by double-clicking or double-tapping.

**IMPORTANT**

The extension may interfere with how some websites handle link clicks (including sites that use custom JavaScript navigation). This prevents unwanted side effects while still allowing you to use the double-tap-to-copy feature when needed.
